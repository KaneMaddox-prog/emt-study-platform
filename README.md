# RescueIQ — EMT Certification Study Platform

An AI-powered study platform for EMT students preparing for the NREMT certification exam.

## Features
- AI-generated NREMT-style practice questions by exam domain
- Interactive patient care scenario trainer with branching decisions
- Student and instructor dashboards with role-based access
- Aligned to the 2025 NREMT EMT Certification Examination Test Plan

## Tech Stack
React, Tailwind CSS, Supabase, Anthropic API, Vercel

## Status
Active development — core quiz flow live, auth system complete, instructor tools in progress.
